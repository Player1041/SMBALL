# Super Monkey Ball Achievement (and) Leaderboard Library (SMBALL)

## What is it?
This is a library created to make creating achievements for Super Monkey Ball 2 a *lot* easier by providing common functions and achievement templates, so all you have to do is provide the correct addresses.
